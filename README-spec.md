# Project: Vian's Growth Portfolio (The "High-Signal" Resume)

## 1. Visual Identity
- **Style:** Minimalist "Indie Hacker" (Ref: stephsmith.io).
- **Typography:** Serif for headings (trust/authority), Sans-serif for body (readability).
- **Colors:** Monochrome base with a single accent color (e.g., Electric Blue) for CTAs.

## 2. Page Structure (Single Page Scroll)
- **Hero:** Punchy value prop ("I build growth engines for SE Asia's top tech brands") + PDF Download Link.
- **The "Big 3" Results:** Visual cards for StashAway, Carsome, and Shopee.
- **Proof of Work:** A "Signal Scraper" project section (link to your AI agent work).
- **Contact:** Clean footer with LinkedIn and Email.

## 3. Tech Stack
- **Framework:** Astro (Static)
- **Styling:** Tailwind CSS
- **Deployment:** Cloudflare Pages
